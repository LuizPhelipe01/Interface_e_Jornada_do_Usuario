# 📱 Relacionamentos & Vida Social

## 📌 Sobre o projeto

Este projeto foi desenvolvido como uma atividade acadêmica com o objetivo de propor uma **nova funcionalidade para um aplicativo já existente**, focada na organização de eventos sociais.

A solução permite que usuários criem eventos, convidem pessoas e acompanhem confirmações de presença de forma simples e intuitiva.

## 🎯 Objetivo

O principal objetivo do projeto é:

- Criar uma funcionalidade de **organização de eventos sociais**
- Reduzir a desorganização de usuários com pouco tempo disponível.
- Integrar a solução a aplicativos já existentes.
- Melhorar a experiência do usuário.

## 👤 Público-alvo

- Universitários desorganizados  
- Pessoas com rotina corrida  
- Usuários que desejam facilitar a organização de encontros e eventos  

## 🧠 Problema resolvido

Muitas pessoas têm dificuldade em organizar eventos, controlar confirmações e manter tudo centralizado.

Este projeto resolve isso ao:

- Centralizar informações do evento
- Facilitar convites
- Melhorar o controle de confirmações

## ⚙️ Funcionalidades

- 📅 Criação de eventos
- 👥 Convite de participantes
- 📊 Visualização de participantes confirmados
- 🔔 Interface intuitiva integrada ao fluxo do app
